Waves tries to address an issue faced by many tourists. Oftentimes, travellers are forced to resort to social media to decide their next destination; with this software, I aim to mitigate that problem.

Technologies: Python, Flask, Pandas, Front-End, Requests, SQL, APIs

Regarding the backend the main programming language was Python, in particular the most used packages were Pandas, Llama, Requests, Flask and os. For data storage we opted for XLS or CSV when the data concerned any individual aspect of a country, while we used JSON when there was no operational value to having a relational database.
The backend was structured following an MVC (Model–View–Controller) design pattern, which allowed for a clear separation of concerns. Data processing, analysis, and ranking logic were encapsulated within the Model layer, while Flask routes acted as controllers.
